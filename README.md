## Para rodar local
- Clonar
  - ``` git clone [url do projeto] ```
- Instalar dependências
  - ``` npm install ```
- Rodar projeto: 
  - ```npm run dev```

## Tecnologias
- ReactJS + Vite
- VimeJS       
- TailwindCSS
- TypeScript
